Consumer off-the-shelf (COTS) developer relations tooling.

Install

    # Python 3+
    # Node.js 12+
    pip install -r requirements.txt
    npm install

Configure

- Google YouTube API key
- Twitter key
- Twitter secret
- Stack Apps application key

Start

    npm start

Aggregate

    python blog.py
    python dev.py
    python medium.py
    python youtube.py
    python twitter.py
    # python so.py
    # python github.py
    # python reddit.py

Manage

    http://localhost:8000
